Query 8: Double events with same name

Every item in the IISG dataset can link to events for that item. For example, the "meeting notes of the CISL Congresso" can link to the event "CISL Congresso" but maybe also to a different event. The problem arises when an item points to two different events that both have the same name. As shown in the table below. Each item has multiple events linked to the item, but the events are equal as they have the same name. This could make linking attributes to certain events harder as different attributes are can now be linked to both items.

Even though events have the same name, events happened in different years and/or locations, thus the items bundle different events.
